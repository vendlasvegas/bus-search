Bus Number Search

How to Use:

  1. Enter a bus number in the search box above
  2. Click "Search"
  3. When Excel loads, click the filter button in the Data tab if filters aren't showing
  4. Click the filter dropdown on column F (Bus Number)
  5. Type your bus number in the search box or uncheck "Select All" and check only your bus number
  6. Click OK to see only matching rows

Important columns: A (Date), C (Driver), F (Bus Number), I (Location), R (Notes)

Note: You must have permission to access this Excel file.
